Определение сетевого адреса Windows-хоста для WSL2

Данные Windows по интерфейсам WSL2:
  vEthernet (WSL (Hyper-V firewall)): 172.19.208.1

Данные внутри Ubuntu WSL2:
  default via 172.19.208.1 dev eth0 proto kernel 
  172.19.208.0/20 dev eth0 proto kernel scope link src 172.19.219.165 
  ---
  172.19.219.165 

Выбранный IP Windows-хоста для JSON-модели и MAVLink: 172.19.208.1
Причина выбора: этот адрес является адресом интерфейса vEthernet (WSL) в Windows и одновременно адресом шлюза по умолчанию, который видит Ubuntu внутри WSL2.
Итоговые параметры обмена:
  JSON-модель ArduPilot -> MATLAB: 172.19.208.1:9002
  MAVLink ArduPilot -> наземная станция: udp:172.19.208.1:14550
