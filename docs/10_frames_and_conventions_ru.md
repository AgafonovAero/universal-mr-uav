# Системы координат и соглашения

## Базовые соглашения

- земная система координат: `NED`
- связанная система координат: `X` вперед, `Y` вправо, `Z` вниз
- внутренняя ориентация: кватернион
- углы Эйлера допускаются только для отображения, отладки и отдельных интерфейсов
- единицы измерения: только `SI`
- углы в коде: только радианы

## Состояние модели

В текущем ядре используется упакованный вектор состояния:

`x = [p_ned_m; v_b_mps; q_nb; w_b_rps]`

где:

- `p_ned_m` - положение в земной системе координат, м
- `v_b_mps` - линейная скорость в связанной системе координат, м/с
- `q_nb` - scalar-first кватернион поворота из связанной системы координат в `NED`
- `w_b_rps` - угловая скорость в связанной системе координат, рад/с

## Силы и моменты

- вход математической модели движения задается силами и моментами в связанной системе координат
- положительная тяга ротора трактуется как скалярная величина, направленная против `+Z_b`
- результирующая сила тяги, приложенная к объекту управления, записывается как `[0; 0; -T]`
- гравитация в `NED` записывается как `[0; 0; g]`

## Геометрия quad-X

Для `quad-X` в preset явно используются три разные величины:

- `wheelbase_m` - диагональ между противоположными роторами, motor-to-motor, м
- `motor_radius_m` - расстояние от центра аппарата до ротора, м
- `motor_xy_m` - матрица `4x2` координат роторов в связанной системе координат, м

Нумерация роторов:

- `1` - front-left
- `2` - front-right
- `3` - rear-right
- `4` - rear-left

Структура `motor_xy_m`:

`motor_xy_m(i, :) = [x_i, y_i]`

Для симметричного `quad-X` выполняются связи:

- `motor_radius_m = wheelbase_m / 2`
- `abs(x_i) = abs(y_i) = motor_radius_m / sqrt(2)`

Матрица микширования использует физические плечи каждого ротора:

- по крену: `Mx = sum(-y_i * T_i)`
- по тангажу: `My = sum(x_i * T_i)`
- по рысканью: `Mz = sum((kQ / kT) * spin_dir(i) * T_i)`

Тем самым в микшере используется именно проекция координаты ротора на соответствующую ось, без повторного деления на `sqrt(2)`.

## Допущения Stage-1

- объект управления рассматривается как твердое тело
- масса и тензор инерции постоянны
- среда включает только гравитацию
- модель ВМГ линейна по квадрату угловой скорости ротора
- `quad-X` рассматривается как единственная схема смешения на этом этапе

Эти соглашения должны сохраняться во всех последующих кодовых расширениях `Stage-1`.
