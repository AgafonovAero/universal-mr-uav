# Область работ Stage-1 / Stage-1.5

Репозиторий формирует text-first и code-centric ядро проекта универсального многороторного БПЛА для MATLAB.

## Назначение этапа

- Зафиксировать математическую модель движения как основной источник истины.
- Построить минимальное расчетное ядро объекта управления без Simulink shell.
- Подготовить основу для последующего расширения модели, тестов и автоматизации.

## Что входит в Stage-1 / Stage-1.5

- объект управления как твердое тело
- математическая модель движения 6DOF на кватернионах
- простая модель ВМГ
- дискретная модель `ESC + motor` первого порядка
- агрегатор сил и моментов в связанной системе координат
- минимальный plant kernel с 17-мерным состоянием
- baseline preset для `quad-X 250 mm`
- demo-сценарии и unit tests
- raw logs и отчеты в репозитории

## Что не входит в текущий этап

- Simulink-модели
- GUI и MLAPP
- бинарные project-файлы
- сложная аэродинамика forward flight
- развитое траекторное управление
- идентификация и валидация по летным испытаниям

## Результат этапа

После завершения `Stage-1.5` репозиторий должен содержать минимальное расчетное ядро, на котором можно прозрачно и воспроизводимо проверять динамику ВМГ, баланс тяги и базовые свойства объекта управления без перехода в `.slx`.
