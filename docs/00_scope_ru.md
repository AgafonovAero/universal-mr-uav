# Область работ Stage-1

Stage-1 формирует тексто-ориентированное ядро проекта универсального многороторного БПЛА для MATLAB.

## Назначение этапа

- Сформировать минимальный кодовый каркас проекта.
- Зафиксировать математическую модель движения как основной источник истины.
- Подготовить базу для последующей интеграции с Simulink без зависимости от GUI.

## Что входит в Stage-1

- объект управления: многороторный БПЛА как твердое тело
- математическая модель движения 6DOF на кватернионах
- простая модель ВМГ по законам `T = kT * omega^2` и `Q = kQ * omega^2`
- земная система координат NED и связанная система координат аппарата
- модель среды только с гравитацией
- минимальный каркас системы автоматического управления по угловым скоростям
- базовый hover demo для проверки баланса тяги и веса
- базовая верификация в виде двух unit tests

## Что не входит в Stage-1

- SLX-модели
- MLAPP-приложения
- бинарные project-файлы
- детальная аэродинамика
- траекторное управление
- контур угловой стабилизации в полном виде
- контур стабилизации высоты
- идентификация и валидация по летным испытаниям

## Результат этапа

После завершения Stage-1 репозиторий должен содержать минимальный, читаемый и расширяемый кодовый каркас, на который можно наращивать более детальную модель, верификацию и средства автоматизации.
